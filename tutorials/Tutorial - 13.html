Q1: Roots of Complex Numbers

Question:

Find the following:

(a) Square Root of $4 + 3i$.

(b) Cube Root of $2 + 3i$.

Final Answer:

(a) $\pm \frac{3+i}{\sqrt{2}}$ or $\pm (\frac{3\sqrt{2}}{2} + i \frac{\sqrt{2}}{2})$.

(b) $13^{1/6} e^{i(\arctan(3/2) + 2k\pi)/3}$ for $k=0, 1, 2$.

Full Solution:

(a) Square Root of $4 + 3i$.
Let $w = a + ib$ such that $w^2 = 4+3i$.
$(a^2 - b^2) + i(2ab) = 4+3i$.
Equating real and imaginary parts: $a^2 - b^2 = 4$ (1) and $2ab = 3$ (2).
Equating moduli squared: $|w|^2 = |4+3i| \implies a^2 + b^2 = \sqrt{4^2+3^2} = 5$ (3).
Add (1) and (3): $2a^2 = 9 \implies a^2 = 9/2 \implies a = \pm 3/\sqrt{2}$.
Subtract (1) from (3): $2b^2 = 1 \implies b^2 = 1/2 \implies b = \pm 1/\sqrt{2}$.
From (2), $ab = 3/2 > 0$, so $a$ and $b$ have the same sign.
The square roots are $\frac{3}{\sqrt{2}} + i \frac{1}{\sqrt{2}}$ and $-\frac{3}{\sqrt{2}} - i \frac{1}{\sqrt{2}}$.

(b) Cube Root of $2 + 3i$.
Let $z = 2+3i$. Find polar form $z=re^{i\theta}$.
$r = |z| = \sqrt{2^2+3^2} = \sqrt{13}$.
$\theta = \text{Arg}(z) = \arctan(3/2)$ (since $z$ is in the first quadrant).
So $z = \sqrt{13} e^{i \arctan(3/2)}$.
The cube roots $w_k$ are given by $w_k = r^{1/3} e^{i(\theta+2k\pi)/3}$ for $k=0,1,2$. $$ w_k = (\sqrt{13})^{1/3} e^{i(\arctan(3/2) + 2k\pi)/3} = 13^{1/6} e^{i(\arctan(3/2) + 2k\pi)/3} $$ The three roots are:
$k=0: w_0 = 13^{1/6} e^{i(\arctan(3/2))/3}$
$k=1: w_1 = 13^{1/6} e^{i(\arctan(3/2) + 2\pi)/3}$
$k=2: w_2 = 13^{1/6} e^{i(\arctan(3/2) + 4\pi)/3}$

Q2: Principal Values of Powers

Question:

Calculate the principal values of $i^i$ and $\pi^i$.

Final Answer:

$i^i = e^{-\pi/2}$

$\pi^i = e^{i \ln \pi} = \cos(\ln \pi) + i \sin(\ln \pi)$

Full Solution:

The principal value of $z^c$ is $e^{c \text{Log}(z)}$, where Log$(z) = \ln|z| + i\text{Arg}(z)$, $-\pi < \text{Arg}(z) \le \pi$.

Principal value of $i^i$:
$z=i$, $c=i$. $|z|=1$. Arg$(z)=\pi/2$.
Log$(i) = \ln(1) + i(\pi/2) = i\pi/2$.
$i^i = e^{i \text{Log}(i)} = e^{i(i\pi/2)} = e^{-\pi/2}$.

Principal value of $\pi^i$:
$z=\pi$, $c=i$. $|z|=\pi$. Arg$(z)=0$.
Log$(\pi) = \ln(\pi) + i(0) = \ln(\pi)$.
$\pi^i = e^{i \text{Log}(\pi)} = e^{i \ln(\pi)} = \cos(\ln \pi) + i \sin(\ln \pi)$.

Q3: Region of Analyticity

Question:

Find the region where the function $f(z) = \frac{\text{Log}(z+4)}{z^2+i}$ is analytic.

Final Answer:

The function is analytic in $\mathbb{C} \setminus \left( (-\infty, -4] \cup \left\{ \frac{1-i}{\sqrt{2}}, \frac{-1+i}{\sqrt{2}} \right\} \right)$.

Full Solution:

The function $f(z)$ is analytic where the numerator and denominator are analytic and the denominator is non-zero.

1. Numerator $g(z) = \text{Log}(z+4)$: The principal logarithm Log$(w)$ is analytic except for $w \le 0$. Let $w = z+4$. The numerator is analytic except when $z+4 \le 0$, which means $z \le -4$. So, the numerator is analytic for $z \notin (-\infty, -4]$.

2. Denominator $h(z) = z^2+i$: This is a polynomial, analytic everywhere.

3. Denominator zero: $z^2+i = 0 \implies z^2 = -i$.
Write $-i$ in polar form: $e^{-i\pi/2}$.
$z^2 = e^{i(-\pi/2 + 2k\pi)}$.
$z = e^{i(-\pi/2 + 2k\pi)/2}$ for $k=0, 1$.
$k=0: z_0 = e^{-i\pi/4} = \frac{1}{\sqrt{2}} - i\frac{1}{\sqrt{2}}$.
$k=1: z_1 = e^{i(3\pi/2)/2} = e^{i3\pi/4} = -\frac{1}{\sqrt{2}} + i\frac{1}{\sqrt{2}}$.

Combining the conditions, $f(z)$ is analytic except on the ray $(-\infty, -4]$ and at the two points $z_0$ and $z_1$.
Region of analyticity: $\mathbb{C} \setminus \left( (-\infty, -4] \cup \{ z_0, z_1 \} \right)$.

Q4: Values of $\sin^{-1} \sqrt{5}$

Question:

Find all the values of $\sin^{-1} \sqrt{5}$.

Final Answer:

$w = (\frac{\pi}{2} + 2n\pi) \pm i \ln(\sqrt{5}+2)$, for $n \in \mathbb{Z}$.

Full Solution:

Let $w = \sin^{-1}(\sqrt{5})$, so $\sin w = \sqrt{5}$. Use the formula: $$ \sin^{-1}(z) = -i \log(iz + \sqrt{1-z^2}) $$ Here $z = \sqrt{5}$.
$1-z^2 = 1 - (\sqrt{5})^2 = 1-5 = -4$.
$\sqrt{1-z^2} = \sqrt{-4} = \pm 2i$.

Case 1: Use $\sqrt{-4} = 2i$. $$ w_1 = -i \log(i\sqrt{5} + 2i) = -i \log(i(\sqrt{5}+2)) $$ Let $\zeta = i(\sqrt{5}+2)$. $|\zeta| = \sqrt{5}+2$. $\arg(\zeta) = \pi/2 + 2k\pi$. $$ \log(\zeta) = \ln(\sqrt{5}+2) + i(\pi/2 + 2k\pi) $$ $$ w_1 = -i [\ln(\sqrt{5}+2) + i(\pi/2 + 2k\pi)] $$ $$ w_1 = -i\ln(\sqrt{5}+2) + (\pi/2 + 2k\pi) $$

Case 2: Use $\sqrt{-4} = -2i$. $$ w_2 = -i \log(i\sqrt{5} - 2i) = -i \log(i(\sqrt{5}-2)) $$ Let $\eta = i(\sqrt{5}-2)$. $|\eta| = \sqrt{5}-2$. $\arg(\eta) = \pi/2 + 2m\pi$. $$ \log(\eta) = \ln(\sqrt{5}-2) + i(\pi/2 + 2m\pi) $$ Note $\ln(\sqrt{5}-2) = \ln(\frac{1}{\sqrt{5}+2}) = -\ln(\sqrt{5}+2)$. $$ \log(\eta) = -\ln(\sqrt{5}+2) + i(\pi/2 + 2m\pi) $$ $$ w_2 = -i [-\ln(\sqrt{5}+2) + i(\pi/2 + 2m\pi)] $$ $$ w_2 = i\ln(\sqrt{5}+2) + (\pi/2 + 2m\pi) $$

Combining both cases, let $n$ be any integer ($k$ or $m$): $$ w = (\frac{\pi}{2} + 2n\pi) \pm i \ln(\sqrt{5}+2) \quad (n \in \mathbb{Z}) $$

Q5: Contour Integrals

Question:

Evaluate $\int_C f(z) dz$, along the indicated contour $C$ :

(a) $f(z) = ze^{z^2}$ and $C$ is path from $0$ to $1+i$ along the parabola $y = x^2$.

(b) $f(z) = \begin{cases} 2 & x < 0 \\ 6x & x > 0 \end{cases}$ and $C$ is the parabola $y = x^2$ from $z = -1 + i$ to $z = 1 + i$.

Final Answer:

(a) $\frac{1}{2}(e^{2i} - 1) = \frac{\cos(2)-1}{2} + i\frac{\sin(2)}{2}$.

(b) $5 + 2i$.

Full Solution:

(a) $f(z) = ze^{z^2}$ and $C: y = x^2$ from $0$ to $1+i$.
$f(z)$ is entire. An antiderivative is $F(z) = \frac{1}{2}e^{z^2}$.
$\int_C f(z) dz = F(1+i) - F(0)$.
$F(0) = \frac{1}{2}e^{0^2} = 1/2$.
$F(1+i) = \frac{1}{2}e^{(1+i)^2} = \frac{1}{2}e^{1+2i-1} = \frac{1}{2}e^{2i}$.
Integral $= \frac{1}{2}e^{2i} - \frac{1}{2} = \frac{1}{2}(\cos 2 + i\sin 2 - 1)$.

(b) $f(z) = \begin{cases} 2 & x < 0 \\ 6x & x > 0 \end{cases}$ and $C: y = x^2$ from $-1+i$ to $1+i$.
Parametrize $C$: $z(t) = t + it^2$, for $-1 \le t \le 1$.
$z'(t) = 1 + 2it$.
On $C$, $x=t$. $f(z(t)) = 2$ for $t \in [-1, 0)$ and $f(z(t)) = 6t$ for $t \in (0, 1]$.
$$ \int_C f(z) dz = \int_{-1}^{1} f(z(t)) z'(t) dt $$ $$ = \int_{-1}^{0} 2 (1+2it) dt + \int_{0}^{1} 6t (1+2it) dt $$ $$ = \int_{-1}^{0} (2 + 4it) dt + \int_{0}^{1} (6t + 12it^2) dt $$ $$ = \left[ 2t + 2it^2 \right]_{-1}^{0} + \left[ 3t^2 + 4it^3 \right]_{0}^{1} $$ $$ = [ (0) - (2(-1)+2i(-1)^2) ] + [ (3(1)^2+4i(1)^3) - (0) ] $$ $$ = -(-2+2i) + (3+4i) = 2-2i + 3+4i = 5+2i $$

Q6: Integrals around a Circle

Question:

Let $C$ be $|z - z_0| = R$ (orientation is counterclock). Obtain

(a) $\int_C \frac{dz}{z-z_0}$

(b) $\int_C (z - z_0)^{n-1} dz \quad n \ne 0$.

Final Answer:

(a) $2\pi i$.

(b) $0$ (for integer $n \ne 0$).

Full Solution:

Parametrize $C$: $z(\theta) = z_0 + Re^{i\theta}$, $0 \le \theta \le 2\pi$. $dz = iRe^{i\theta} d\theta$.

(a) $\int_C \frac{dz}{z-z_0}$ $$ \int_{0}^{2\pi} \frac{iRe^{i\theta} d\theta}{(z_0 + Re^{i\theta}) - z_0} = \int_{0}^{2\pi} \frac{iRe^{i\theta}}{Re^{i\theta}} d\theta = \int_{0}^{2\pi} i d\theta $$ $$ = [i\theta]_0^{2\pi} = i(2\pi) - i(0) = 2\pi i $$

(b) $\int_C (z - z_0)^{n-1} dz \quad n \ne 0$ (integer assumed) $$ \int_{0}^{2\pi} (z_0 + Re^{i\theta} - z_0)^{n-1} (iRe^{i\theta} d\theta) $$ $$ = \int_{0}^{2\pi} (Re^{i\theta})^{n-1} iRe^{i\theta} d\theta = iR^n \int_{0}^{2\pi} e^{i(n-1)\theta} e^{i\theta} d\theta $$ $$ = iR^n \int_{0}^{2\pi} e^{in\theta} d\theta $$ Since $n \ne 0$ is an integer: $$ = iR^n \left[ \frac{e^{in\theta}}{in} \right]_{0}^{2\pi} = \frac{R^n}{n} [e^{i2\pi n} - e^{i0}] = \frac{R^n}{n}[1-1] = 0 $$

Q7: Upper Bound for Integrals (ML-Inequality)

Question:

Find the upper bound for the absolute value of $\int_C f(z) dz$, along the indicated contour $C$:

(a) $f(z) = \frac{1}{z^2+1}$ and $C$ is the line segment from $z = 3$ to $z = 3 + i$.

(b) $f(z) = \text{Log}(z + 3)$ and $C$ is the line segment from $z = 3i$ to $z = 4 + 3i$.

Final Answer:

(a) $\frac{1}{10}$.

(b) $4 \sqrt{(\frac{1}{2}\ln 58)^2 + (\arctan(3/7))^2}$.

Full Solution:

Use ML-inequality: $|\int_C f(z) dz| \le ML$, where $L=$ Length(C) and $M \ge |f(z)|$ on C.

(a) $f(z) = \frac{1}{z^2+1}$, $C$: segment from 3 to 3+i.
Length $L = |(3+i) - 3| = |i| = 1$.
Need $M$ s.t. $|\frac{1}{z^2+1}| \le M$. Find $\min |z^2+1|$ on $C$.
Let $z=3+it$, $0 \le t \le 1$. $|z^2+1|^2 = |(10-t^2)+6it|^2 = (10-t^2)^2 + 36t^2 = t^4 + 16t^2 + 100$.
This is increasing for $t \ge 0$. Minimum occurs at $t=0$.
$\min |z^2+1|^2 = 100$. $\min |z^2+1| = 10$.
So, $|f(z)| = 1/|z^2+1| \le 1/10 = M$.
Bound: $ML = \frac{1}{10} \cdot 1 = \frac{1}{10}$.

(b) $f(z) = \text{Log}(z + 3)$, $C$: segment from 3i to 4+3i.
Length $L = |(4+3i) - 3i| = |4| = 4$.
Need $M$ s.t. $|\text{Log}(z+3)| \le M$. Find $\max |\text{Log}(z+3)|$ on $C$.
Let $z=t+3i$, $0 \le t \le 4$. $z+3 = (t+3)+3i$.
$|\text{Log}(z+3)| = |\ln|z+3| + i\text{Arg}(z+3)| = \sqrt{(\ln|z+3|)^2 + (\text{Arg}(z+3))^2}$.
$|z+3| = \sqrt{(t+3)^2+9}$. $\text{Arg}(z+3) = \arctan(\frac{3}{t+3})$.
Let $g(t) = |\text{Log}(z(t)+3)|^2 = (\frac{1}{2}\ln((t+3)^2+9))^2 + (\arctan(\frac{3}{t+3}))^2$.
The term $\ln(\dots)$ increases as $t$ increases. The term $\arctan(\dots)$ decreases as $t$ increases.
Compare endpoints:
$t=0: g(0) = (\frac{1}{2}\ln 18)^2 + (\arctan 1)^2 = (\frac{1}{2}\ln 18)^2 + (\pi/4)^2 \approx 2.71$.
$t=4: g(4) = (\frac{1}{2}\ln 58)^2 + (\arctan(3/7))^2 \approx 4.28$.
The maximum occurs at $t=4$. $M = \sqrt{g(4)} = \sqrt{(\frac{1}{2}\ln 58)^2 + (\arctan(3/7))^2}$.
Bound: $ML = 4 \sqrt{(\frac{1}{2}\ln 58)^2 + (\arctan(3/7))^2}$.

Q8: Integral of $3z^{1/2}$

Question:

Find the value of $\int_C 3z^{1/2} dz$, where $C$ is the line segment between $z_0 = 1$ and $z_1 = 9i$ and $z^{1/2}$ is the principal branch of the square root function.

Final Answer:

$-(2 + 27\sqrt{2}) + i(27\sqrt{2})$

Full Solution:

The function $f(z) = 3z^{1/2}$ (principal branch) is analytic except on $(-\infty, 0]$. The contour $C$ from $1$ to $9i$ does not cross the branch cut.
An antiderivative is $F(z) = 2z^{3/2}$. We use the principal value: $z^{3/2} = |z|^{3/2} e^{i(3/2)\text{Arg}(z)}$.
The integral is $F(z_1) - F(z_0) = F(9i) - F(1)$.

Calculate $F(9i)$: $|9i|=9$, Arg$(9i)=\pi/2$. $$ F(9i) = 2 (9i)^{3/2} = 2 |9i|^{3/2} e^{i(3/2)(\pi/2)} = 2 (9^{3/2}) e^{i3\pi/4} $$ $$ = 2 (3^3) (\cos(3\pi/4) + i\sin(3\pi/4)) = 54 (-\frac{1}{\sqrt{2}} + i\frac{1}{\sqrt{2}}) $$ $$ = 27\sqrt{2}(-1+i) = -27\sqrt{2} + 27\sqrt{2}i $$

Calculate $F(1)$: $|1|=1$, Arg$(1)=0$. $$ F(1) = 2 (1)^{3/2} = 2 |1|^{3/2} e^{i(3/2)(0)} = 2(1)e^0 = 2 $$

The integral is: $$ F(9i) - F(1) = (-27\sqrt{2} + 27\sqrt{2}i) - 2 $$ $$ = -(2 + 27\sqrt{2}) + i(27\sqrt{2}) $$
