Q1: Triangle & Reverse Triangle Inequality

Question:

Show that $||z| - |w|| \le |z - w| \le |z| + |w|$.

Final Answer:

The inequalities $||z| - |w|| \le |z - w|$ (reverse triangle inequality) and $|z - w| \le |z| + |w|$ (triangle inequality) are proven using properties of complex numbers and moduli.

Full Solution:

We need to prove two inequalities: (i) $|z - w| \le |z| + |w|$
(ii) $||z| - |w|| \le |z - w|$

(i) Proof of $|z - w| \le |z| + |w|$
This is a direct application of the triangle inequality, which states that for any complex numbers $z_1$ and $z_2$, we have $|z_1 + z_2| \le |z_1| + |z_2|$.
Let $z_1 = z$ and $z_2 = -w$. Applying the triangle inequality: $$|z + (-w)| \le |z| + |-w|$$ $$|z - w| \le |z| + |w|$$ (since $|-w| = |-1||w| = 1 \cdot |w| = |w|$)
This proves the second part of the inequality.

(ii) Proof of $||z| - |w|| \le |z - w|$
This is known as the reverse triangle inequality.
From the triangle inequality proven above, we have: $$|z| = |(z - w) + w| \le |z - w| + |w|$$ Subtracting $|w|$ from both sides gives: $$|z| - |w| \le |z - w| \quad (1)$$ Also, we can write: $$|w| = |(w - z) + z| \le |w - z| + |z|$$ Since $|w - z| = |-(z - w)| = |-1||z - w| = |z - w|$, we have: $$|w| \le |z - w| + |z|$$ Subtracting $|z|$ from both sides gives: $$|w| - |z| \le |z - w|$$ Multiplying by -1 reverses the inequality sign: $$-(|w| - |z|) \ge -|z - w|$$ $$|z| - |w| \ge -|z - w| \quad (2)$$ Combining Equation (1) and Equation (2):
We have $-|z - w| \le |z| - |w| \le |z - w|$.
This is the definition of the absolute value, so we can write: $$||z| - |w|| \le |z - w|$$

Combining both parts, we have shown that $||z| - |w|| \le |z - w| \le |z| + |w|$.

Q2: Generalized Triangle Inequality (Induction)

Question:

Prove, by induction, that $|z_1 + z_2 + \dots + z_n| \le |z_1| + |z_2| + \dots + |z_n|$.

Final Answer:

The generalized triangle inequality $|z_1 + \dots + z_n| \le |z_1| + \dots + |z_n|$ is proven for $n \ge 2$ using mathematical induction, based on the standard triangle inequality for $n=2$.

Full Solution:

We want to prove the generalized triangle inequality using mathematical induction. Let $P(n)$ be the statement $|z_1 + z_2 + \dots + z_n| \le |z_1| + |z_2| + \dots + |z_n|$.

Base Case: For $n=2$, the statement is $P(2): |z_1 + z_2| \le |z_1| + |z_2|$. This is the standard triangle inequality for complex numbers, which we assume to be true. (The case $n=1$ is $|z_1| \le |z_1|$, which is trivial).

Inductive Hypothesis: Assume that $P(k)$ is true for some integer $k \ge 2$. That is, assume: $$|z_1 + z_2 + \dots + z_k| \le |z_1| + |z_2| + \dots + |z_k|$$

Inductive Step: We need to show that $P(k+1)$ is true, i.e., $$|z_1 + z_2 + \dots + z_k + z_{k+1}| \le |z_1| + |z_2| + \dots + |z_k| + |z_{k+1}|$$

Consider the left side of the inequality for $P(k+1)$: $|z_1 + z_2 + \dots + z_k + z_{k+1}|$
Group the first $k$ terms: $|(z_1 + z_2 + \dots + z_k) + z_{k+1}|$
Apply the base case triangle inequality ($|A + B| \le |A| + |B|$) with $A = z_1 + z_2 + \dots + z_k$ and $B = z_{k+1}$: $$|(z_1 + z_2 + \dots + z_k) + z_{k+1}| \le |z_1 + z_2 + \dots + z_k| + |z_{k+1}|$$ Now, apply the inductive hypothesis $|z_1 + z_2 + \dots + z_k| \le |z_1| + |z_2| + \dots + |z_k|$ to the first term on the right side: $$|z_1 + z_2 + \dots + z_k| + |z_{k+1}| \le (|z_1| + |z_2| + \dots + |z_k|) + |z_{k+1}|$$ Combining the steps: $$|z_1 + z_2 + \dots + z_k + z_{k+1}| \le |z_1| + |z_2| + \dots + |z_k| + |z_{k+1}|$$ This is the statement $P(k+1)$.

Conclusion: Since the base case $P(2)$ is true and we have shown that $P(k)$ implies $P(k+1)$ for $k \ge 2$, by the principle of mathematical induction, the inequality $|z_1 + z_2 + \dots + z_n| \le |z_1| + |z_2| + \dots + |z_n|$ holds for all integers $n \ge 2$.

Q3: Sketching Subsets of $\mathbb{C}$

Question:

Sketch the following subsets of the complex plane:
(a) $\{z : |z| < 1\}$
(b) $\{z : \text{Re}(z) < 1\}$
(c) $\{z : \text{Re}(z) = 1\}$
(d) $\{z : |z - 1| < |z + 1|\}$
(e) $\{z : 0 < |z - 1| < 2\}$
(f) $\{z : |2z + i| = 4\}$
(g) $\{z : |z - 4| \ge |z|\}$

Final Answer:

(a) Open disk, radius 1, center origin.
(b) Open left half-plane $x < 1$.
(c) Vertical line $x = 1$.
(d) Open right half-plane $x > 0$.
(e) Open disk, radius 2, center $z=1$, punctured at $z=1$.
(f) Circle, radius 2, center $z=-i/2$.
(g) Closed left half-plane $x \le 2$.

Full Solution:

Let $z = x + iy$, where $x = \text{Re}(z)$ and $y = \text{Im}(z)$.

(a) $\{z : |z| < 1\}$
$|z| = \sqrt{x^2 + y^2}$. The condition is $\sqrt{x^2 + y^2} < 1$, or $x^2 + y^2 < 1^2$.
This is the interior of the circle centered at the origin (0,0) with radius 1. The boundary is not included.
Sketch description: An open disk centered at the origin with radius 1.

(b) $\{z : \text{Re}(z) < 1\}$
$\text{Re}(z) = x$. The condition is $x < 1$.
This is the set of all points to the left of the vertical line $x=1$. The line $x=1$ is not included.
Sketch description: The open left half-plane defined by $x < 1$.

(c) $\{z : \text{Re}(z) = 1\}$
$\text{Re}(z) = x$. The condition is $x = 1$.
This is the set of all points on the vertical line $x=1$.
Sketch description: The vertical line passing through the point (1,0).

(d) $\{z : |z - 1| < |z + 1|\}$
This condition means the distance from $z$ to 1 is less than the distance from $z$ to -1. This defines the open half-plane to the right of the perpendicular bisector of the segment from -1 to 1. The bisector is the imaginary axis ($x=0$).
Algebraically: $|(x - 1) + iy| < |(x + 1) + iy|$ $\implies \sqrt{(x - 1)^2 + y^2} < \sqrt{(x + 1)^2 + y^2}$ $\implies (x - 1)^2 + y^2 < (x + 1)^2 + y^2$ $\implies x^2 - 2x + 1 < x^2 + 2x + 1$ $\implies -2x < 2x \implies 0 < 4x \implies x > 0$.
This is $\text{Re}(z) > 0$.
Sketch description: The open right half-plane defined by $x > 0$.

(e) $\{z : 0 < |z - 1| < 2\}$
$|z - 1|$ is the distance from $z$ to 1.
$|z - 1| < 2$ means inside the circle of radius 2 centered at $z=1$.
$|z - 1| > 0$ means $z \ne 1$.
This is the set of points inside the circle of radius 2 centered at 1, excluding the center point $z=1$.
Sketch description: An open disk centered at $z=1$ with radius 2, with the center point $z=1$ removed (a punctured disk).

(f) $\{z : |2z + i| = 4\}$
$|2z + i| = |2(z + i/2)| = 2 |z - (-i/2)|$.
The equation is $2 |z - (-i/2)| = 4$, which simplifies to $|z - (-i/2)| = 2$.
This is the set of points $z$ whose distance from $-i/2$ is exactly 2.
Sketch description: A circle centered at $z_0 = -i/2$ (or $(0, -1/2)$) with radius $R=2$.

(g) $\{z : |z - 4| \ge |z|\}$
This condition means the distance from $z$ to 4 is greater than or equal to the distance from $z$ to 0. The boundary is the perpendicular bisector $x=2$. The region includes points closer to 0 or equidistant.
Algebraically: $|(x - 4) + iy| \ge |x + iy|$ $\implies \sqrt{(x - 4)^2 + y^2} \ge \sqrt{x^2 + y^2}$ $\implies (x - 4)^2 + y^2 \ge x^2 + y^2$ $\implies x^2 - 8x + 16 \ge x^2$ $\implies -8x + 16 \ge 0 \implies 16 \ge 8x \implies 2 \ge x$.
This is $\text{Re}(z) \le 2$.
Sketch description: The closed left half-plane defined by $x \le 2$, including the boundary line $x=2$.

Q5: Alternative Equation of a Circle

Question:

Show that the equation $|z - z_0| = R$ of a circle, centered at $z_0$ with radius R, can be written as $|z|^2 - 2 \text{Re} (z\bar{z_0}) + |z_0|^2 = R^2$.

Final Answer:

By squaring $|z - z_0| = R$ and using properties $|w|^2 = w\bar{w}$ and $w + \bar{w} = 2\text{Re}(w)$, the equivalent form $|z|^2 - 2 \text{Re} (z\bar{z_0}) + |z_0|^2 = R^2$ is derived.

Full Solution:

Start with the equation of the circle: $$|z - z_0| = R$$ Since both sides are non-negative, we can square both sides: $$|z - z_0|^2 = R^2$$ Use the property that for any complex number $w$, $|w|^2 = w\bar{w}$. Let $w = z - z_0$. $$(z - z_0)(\overline{z - z_0}) = R^2$$ Using properties of conjugates, $\overline{z - z_0} = \bar{z} - \bar{z_0}$. $$(z - z_0)(\bar{z} - \bar{z_0}) = R^2$$ Expand the product on the left side: $$z\bar{z} - z\bar{z_0} - z_0\bar{z} + z_0\bar{z_0} = R^2$$ Recall that $|z|^2 = z\bar{z}$ and $|z_0|^2 = z_0\bar{z_0}$. Substitute these into the equation: $$|z|^2 - z\bar{z_0} - z_0\bar{z} + |z_0|^2 = R^2$$ Now consider the middle terms: $- z\bar{z_0} - z_0\bar{z}$.
Notice that $z_0\bar{z} = \overline{z\bar{z_0}}$.
So, the terms are $-(z\bar{z_0} + \overline{z\bar{z_0}})$.
Recall the property that for any complex number $w$, $w + \bar{w} = 2 \text{Re}(w)$. Let $w = z\bar{z_0}$.
Therefore, $z\bar{z_0} + \overline{z\bar{z_0}} = 2 \text{Re}(z\bar{z_0})$.
Substitute this back into the equation: $$|z|^2 - 2 \text{Re}(z\bar{z_0}) + |z_0|^2 = R^2$$ This is the desired form.

Q6: Principal Argument of $(\sqrt{3} - i)^6$

Question:

Find the principle argument of $z = (\sqrt{3} - i)^6$.

Final Answer:

The principal argument is $\text{Arg}(z) = \pi$.

Full Solution:

First, let's find the polar form of the complex number $w = \sqrt{3} - i$.
The modulus is $r = |w| = \sqrt{(\sqrt{3})^2 + (-1)^2} = \sqrt{3 + 1} = \sqrt{4} = 2$.
To find the argument $\theta = \arg(w)$, we look for $\theta$ such that $\cos \theta = \frac{\sqrt{3}}{2}$ and $\sin \theta = \frac{-1}{2}$.
The angle lies in the fourth quadrant. The principal value $\text{Arg}(w)$ is $\theta = -\frac{\pi}{6}$.
So, $w = 2 (\cos(-\pi/6) + i\sin(-\pi/6)) = 2e^{-i\pi/6}$.

Now, we need to compute $z = w^6$: $$z = (2e^{-i\pi/6})^6$$ Using De Moivre's Theorem $(re^{i\theta})^n = r^n e^{in\theta}$: $$z = 2^6 e^{i(-\pi/6 \times 6)}$$ $$z = 64 e^{-i\pi}$$ The complex number $z$ is $64(\cos(-\pi) + i\sin(-\pi)) = 64(-1 + i(0)) = -64$.
The principal argument, denoted $\text{Arg}(z)$, is the unique argument $\theta$ such that $-\pi < \theta \le \pi$.
For $z = -64$, the point lies on the negative real axis. The angle measured counterclockwise from the positive real axis is $\pi$.
Since $\pi$ is in the interval $(-\pi, \pi]$, the principal argument is $\pi$. $$\text{Arg}((\sqrt{3} - i)^6) = \pi$$

Q7: Cube Roots of $-8i$

Question:

Find all the values of $(-8i)^{1/3}$.

Final Answer:

The three cube roots are $\sqrt{3} - i$, $2i$, and $-\sqrt{3} - i$.

Full Solution:

We want to find the cube roots of the complex number $z = -8i$.
First, write $z$ in polar form $z = re^{i\theta}$.
The modulus is $r = |-8i| = \sqrt{0^2 + (-8)^2} = \sqrt{64} = 8$.
The argument $\theta$ corresponds to the point $(0, -8)$, which lies on the negative imaginary axis. The principal argument is $\text{Arg}(z) = -\frac{\pi}{2}$.
So, the polar form is $z = 8e^{-i\pi/2}$.
More generally, we can write $z = 8e^{i(-\pi/2 + 2k\pi)}$ for any integer $k$.

We are looking for values $w$ such that $w^3 = z$. Let $w = \rho e^{i\phi}$. Then $w^3 = \rho^3 e^{i3\phi}$. $$\rho^3 e^{i3\phi} = 8e^{i(-\pi/2 + 2k\pi)}$$ Equating the moduli: $\rho^3 = 8 \implies \rho = \sqrt[3]{8} = 2$.
Equating the arguments: $3\phi = -\frac{\pi}{2} + 2k\pi$ $$\phi = \frac{-\pi/2 + 2k\pi}{3} = -\frac{\pi}{6} + \frac{2k\pi}{3}$$ We find distinct roots by taking $k = 0, 1, 2$.

For $k=0$:
$\phi_0 = -\frac{\pi}{6}$.
$w_0 = 2e^{i(-\pi/6)} = 2(\cos(-\pi/6) + i\sin(-\pi/6)) = 2(\frac{\sqrt{3}}{2} - i\frac{1}{2}) = \sqrt{3} - i$.

For $k=1$:
$\phi_1 = -\frac{\pi}{6} + \frac{2\pi}{3} = -\frac{\pi}{6} + \frac{4\pi}{6} = \frac{3\pi}{6} = \frac{\pi}{2}$.
$w_1 = 2e^{i(\pi/2)} = 2(\cos(\pi/2) + i\sin(\pi/2)) = 2(0 + i(1)) = 2i$.

For $k=2$:
$\phi_2 = -\frac{\pi}{6} + \frac{4\pi}{3} = -\frac{\pi}{6} + \frac{8\pi}{6} = \frac{7\pi}{6}$.
$w_2 = 2e^{i(7\pi/6)} = 2(\cos(7\pi/6) + i\sin(7\pi/6)) = 2(-\frac{\sqrt{3}}{2} - i\frac{1}{2}) = -\sqrt{3} - i$.

The three cube roots of $-8i$ are $\sqrt{3} - i$, $2i$, and $-\sqrt{3} - i$.

Q8: Accumulation Points of Sets

Question:

Determine the accumulation points for each of the following sets:
(a) $z_n = i^n, n = 1, 2, 3, \dots$
(b) $z_n = i^n/n, n = 1, 2, 3, \dots$
(c) $S = \{z : 0 \le \text{Arg}(z) < \pi/2, z \ne 0\}$
(d) $z_n = (-1)^n (1 + i) \frac{n-1}{n}, n = 1, 2, 3, \dots$

Final Answer:

(a) None.
(b) $\{0\}$.
(c) $\{z : 0 \le \text{Arg}(z) \le \pi/2\}$ (the closed first quadrant).
(d) $\{1+i, -1-i\}$.

Full Solution:

An accumulation point (or limit point) $p$ of a set $S$ is a point such that every neighborhood of $p$ contains at least one point of $S$ distinct from $p$.

(a) $z_n = i^n, n = 1, 2, 3, \dots$
The set of values is $S = \{i, -1, -i, 1\}$.
This is a finite set. A finite set has no accumulation points because for any point $p$, we can find a small enough neighborhood around $p$ that contains no *other* point from the set.
Accumulation points: None.

(b) $z_n = i^n/n, n = 1, 2, 3, \dots$
The set of values is $S = \{i/1, -1/2, -i/3, 1/4, \dots\}$.
Consider the modulus $|z_n| = |i^n/n| = 1/n$. As $n \to \infty$, $|z_n| \to 0$. The sequence converges to 0.
Any neighborhood $B(0, \epsilon)$ contains points $z_n$ for $n > 1/\epsilon$. Since $z_n \ne 0$, 0 is an accumulation point.
Since the sequence converges to 0, no other point $p \ne 0$ can be an accumulation point.
Accumulation points: $\{0\}$.

(c) $S = \{z : 0 \le \text{Arg}(z) < \pi/2, z \ne 0\}$
This set is the first quadrant, excluding the origin and the positive imaginary axis, but including the positive real axis.
Any point $p$ in $S$ is an accumulation point.
The origin $p=0$ is an accumulation point since any $B(0, \epsilon)$ contains points like $\epsilon/2 \in S$.
Any point $p=iy$ on the positive imaginary axis ($y>0$) is an accumulation point since any $B(p, \epsilon)$ contains points like $\delta + iy \in S$ for small $\delta > 0$.
The set of accumulation points is the closure of $S$, which is the closed first quadrant.
Accumulation points: $\{z : 0 \le \text{Arg}(z) \le \pi/2\}$.

(d) $z_n = (-1)^n (1 + i) \frac{n-1}{n}, n = 1, 2, 3, \dots$
Let $w = 1+i$. $z_n = (-1)^n w (1 - 1/n)$.
Subsequence for even $n=2k$: $z_{2k} = w(1 - 1/(2k)) \to w = 1+i$ as $k \to \infty$.
Subsequence for odd $n=2k+1$: $z_{2k+1} = -w(1 - 1/(2k+1)) \to -w = -1-i$ as $k \to \infty$.
The limits of the convergent subsequences are the accumulation points.
Accumulation points: $\{1+i, -1-i\}$.

Q9: Points in a Domain are Accumulation Points

Question:

Show that any point $z_0$ of a domain is an accumulation point of that domain.

Final Answer:

Since a domain $D$ is open, for any $z_0 \in D$, any neighborhood $N$ of $z_0$ contains an open disk $B(z_0, r) \subseteq D$. We can always find a point $z_1 = z_0 + r/2$ within this disk ($z_1 \in D, z_1 \in N$) such that $z_1 \ne z_0$. Thus $z_0$ is an accumulation point.

Full Solution:

Let $D$ be a domain in the complex plane. By definition, a domain is a non-empty, open, and connected set.
Let $z_0$ be an arbitrary point in $D$ ($z_0 \in D$).
We want to show that $z_0$ is an accumulation point of $D$. This means we need to show that every neighborhood of $z_0$ contains at least one point of $D$ that is different from $z_0$.

Let $N$ be any neighborhood of $z_0$. By the definition of a neighborhood, there exists an open disk centered at $z_0$ with some radius $\delta > 0$, denoted $B(z_0, \delta) = \{z \in \mathbb{C} : |z - z_0| < \delta\}$, such that $B(z_0, \delta) \subseteq N$.

Since $D$ is a domain, it is an open set. Because $z_0 \in D$ and $D$ is open, there exists an open disk centered at $z_0$ with some radius $\epsilon > 0$, denoted $B(z_0, \epsilon) = \{z \in \mathbb{C} : |z - z_0| < \epsilon\}$, such that $B(z_0, \epsilon) \subseteq D$.

Now consider the radius $r = \min(\delta, \epsilon)$. Since both $\delta > 0$ and $\epsilon > 0$, we have $r > 0$.
The open disk $B(z_0, r)$ is contained within both $N$ (since $r \le \delta$) and $D$ (since $r \le \epsilon$).
So, $B(z_0, r) \subseteq N$ and $B(z_0, r) \subseteq D$.

Consider the point $z_1 = z_0 + \frac{r}{2}$.
The distance from $z_1$ to $z_0$ is $|z_1 - z_0| = |(z_0 + r/2) - z_0| = |r/2| = r/2$.
Since $r > 0$, we have $0 < r/2 < r$.
Therefore, $z_1$ is in the open disk $B(z_0, r)$.
Since $B(z_0, r) \subseteq D$, we have $z_1 \in D$.
Since $B(z_0, r) \subseteq N$, we have $z_1 \in N$.
Also, since $r/2 \ne 0$, the point $z_1 = z_0 + r/2$ is different from $z_0$.

So, for any neighborhood $N$ of $z_0$, we have found a point $z_1 \in N$ such that $z_1 \in D$ and $z_1 \ne z_0$.
By definition, this means $z_0$ is an accumulation point of $D$.
Since $z_0$ was an arbitrary point in $D$, every point of a domain is an accumulation point of that domain.

Q10: Finite Sets Have No Accumulation Points

Question:

Prove that a finite set cannot have any accumulation points.

Final Answer:

For any point $p$ (either in the finite set $S$ or not), the minimum distance $d$ to other points in $S$ (or all points if $p \notin S$) is positive. The neighborhood $B(p, d)$ contains no point of $S$ different from $p$, so $p$ cannot be an accumulation point.

Full Solution:

Let $S = \{z_1, z_2, \dots, z_n\}$ be a finite set in the complex plane, where $n$ is a finite non-negative integer.
We want to show that no point $p \in \mathbb{C}$ can be an accumulation point of $S$.
Recall that $p$ is an accumulation point of $S$ if every neighborhood of $p$ contains at least one point of $S$ *different from* $p$.

We consider two cases for an arbitrary point $p \in \mathbb{C}$:

Case 1: $p$ is one of the points in the set $S$.
Suppose $p = z_k$ for some $k \in \{1, 2, \dots, n\}$.
Consider the distances between $p$ and all *other* points in $S$: $d_j = |p - z_j|$ for $j \in \{1, \dots, n\}$ and $j \ne k$.
If $n \le 1$, there are no other points, so any neighborhood $B(p, \epsilon)$ contains no points of $S$ other than $p$. Thus, $p$ is not an accumulation point.
If $n > 1$, since all $z_j$ are distinct for $j \ne k$, all $d_j > 0$.
Let $d = \min \{d_j : j \ne k\}$. Since this is a minimum of a finite set of positive numbers, $d > 0$.
Now consider the open disk (neighborhood) $N = B(p, d) = \{z \in \mathbb{C} : |z - p| < d\}$.
This neighborhood $N$ contains $p$.
For any $z_j \in S$ with $j \ne k$, we have $|z_j - p| = d_j \ge d$. Thus, $z_j \notin N$.
The neighborhood $N = B(p, d)$ contains no point of $S$ other than $p$ itself.
Therefore, $p$ is not an accumulation point of $S$.

Case 2: $p$ is not a point in the set $S$.
Suppose $p \notin S$.
Consider the distances between $p$ and *all* points in $S$: $d_j = |p - z_j|$ for $j \in \{1, 2, \dots, n\}$.
Since $p \notin S$, $p \ne z_j$ for all $j$, so all $d_j > 0$. (If $n=0$, $S$ is empty, trivially no accumulation points).
Let $d' = \min \{d_j : j = 1, \dots, n\}$. Since this is a minimum of a finite set of positive numbers, $d' > 0$.
Now consider the open disk (neighborhood) $N' = B(p, d') = \{z \in \mathbb{C} : |z - p| < d'\}$.
For any $z_j \in S$, we have $|z_j - p| = d_j \ge d'$. Thus, $z_j \notin N'$.
The neighborhood $N' = B(p, d')$ contains no points from $S$ at all.
Therefore, $p$ cannot be an accumulation point of $S$.

Conclusion: In both cases, any point $p \in \mathbb{C}$ is not an accumulation point of the finite set $S$. Therefore, a finite set has no accumulation points.
