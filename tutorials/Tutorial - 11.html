Q1: CR Equations for $f^2(z)$

Question:

Show that if $f(z) = u + iv$ satisfies the Cauchy-Riemann equations, then $f^2(z)$ also satisfies the same.

Final Answer:

By setting $f^2(z) = U+iV = (u^2-v^2) + i(2uv)$ and computing the partial derivatives $U_x, U_y, V_x, V_y$, then substituting the CR equations for $u,v$ ($u_x=v_y, u_y=-v_x$), it is verified that $U_x = V_y$ and $U_y = -V_x$.

Full Solution:

Let $f(z) = u(x,y) + iv(x,y)$. Since $f(z)$ satisfies the Cauchy-Riemann (CR) equations, we have: $$u_x = v_y \quad (1)$$ $$u_y = -v_x \quad (2)$$ We assume that $u$ and $v$ have continuous first partial derivatives.

Now consider $f^2(z)$. Let $f^2(z) = U(x,y) + iV(x,y)$. $$f^2(z) = (u+iv)^2 = u^2 + 2iuv + (iv)^2 = (u^2 - v^2) + i(2uv)$$ So, $U = u^2 - v^2$ and $V = 2uv$.

We need to show that $U$ and $V$ satisfy the CR equations, i.e., $U_x = V_y$ and $U_y = -V_x$.
Compute the partial derivatives: $$U_x = \frac{\partial}{\partial x}(u^2 - v^2) = 2u u_x - 2v v_x$$ $$U_y = \frac{\partial}{\partial y}(u^2 - v^2) = 2u u_y - 2v v_y$$ $$V_x = \frac{\partial}{\partial x}(2uv) = 2(u_xv + uv_x)$$ $$V_y = \frac{\partial}{\partial y}(2uv) = 2(u_yv + uv_y)$$

Check the first CR equation for $U, V$: $U_x = V_y$.
Using (1) and (2) in $U_x$: $$U_x = 2u(v_y) - 2v(-u_y) = 2uv_y + 2vu_y$$ The expression for $V_y$ is: $$V_y = 2(u_yv + uv_y) = 2u_yv + 2uv_y$$ Comparing, we see $U_x = V_y$.

Check the second CR equation for $U, V$: $U_y = -V_x$.
Using (1) and (2) in $U_y$: $$U_y = 2u(-v_x) - 2v(u_x) = -2uv_x - 2vu_x$$ The expression for $-V_x$ is: $$-V_x = -2(u_xv + uv_x) = -2u_xv - 2uv_x$$ Comparing, we see $U_y = -V_x$.

Since both CR equations ($U_x = V_y$ and $U_y = -V_x$) are satisfied, $f^2(z)$ also satisfies the Cauchy-Riemann equations.

Q2: Showing Functions are Nowhere Differentiable

Question:

Use Cauchy-Riemann (CR) equations to show that the following functions are nowhere differentiable.

(a) $f(z) = z - \bar{z}$

(b) $f(z) = e^x e^{-iy}$

Final Answer:

(a) $u_x=0, v_y=2$. Since $u_x \ne v_y$, $f$ is nowhere differentiable.

(b) C-R equations require $e^x \cos y = -e^x \cos y$ and $-e^x \sin y = e^x \sin y$. This implies $\cos y = 0$ and $\sin y = 0$, which is impossible. Thus $f$ is nowhere differentiable.

Full Solution:

(a) $f(z) = z - \bar{z}$
Let $z = x + iy$. $f(z) = (x+iy) - (x-iy) = 2iy$.
$u(x,y) = 0$, $v(x,y) = 2y$.
$u_x = 0$, $u_y = 0$.
$v_x = 0$, $v_y = 2$.
Check CR equations:
1. $u_x = v_y \implies 0 = 2$ (False).
2. $u_y = -v_x \implies 0 = -0$ (True).
Since the first CR equation is never satisfied, $f$ is nowhere differentiable.

(b) $f(z) = e^x e^{-iy}$
$f(z) = e^x (\cos(-y) + i\sin(-y)) = e^x (\cos y - i\sin y) = e^x \cos y - i e^x \sin y$.
$u(x,y) = e^x \cos y$, $v(x,y) = -e^x \sin y$.
$u_x = e^x \cos y$, $u_y = -e^x \sin y$.
$v_x = -e^x \sin y$, $v_y = -e^x \cos y$.
Check CR equations:
1. $u_x = v_y \implies e^x \cos y = -e^x \cos y \implies 2e^x \cos y = 0 \implies \cos y = 0$.
2. $u_y = -v_x \implies -e^x \sin y = -(-e^x \sin y) = e^x \sin y \implies 2e^x \sin y = 0 \implies \sin y = 0$.
We need both $\cos y = 0$ and $\sin y = 0$. Since $\cos^2 y + \sin^2 y = 1$, this is impossible.
The CR equations are never satisfied, so $f$ is nowhere differentiable.

Q3: Where $f'(z)$ Exists

Question:

Determine where $f'(z)$ exists and find its value when

(a) $f(z) = x^2 + iy^2$

(b) $f(z) = z \text{Im}(z)$

Final Answer:

(a) Differentiable only on the line $y=x$. $f'(x+ix) = 2x$.

(b) Differentiable only at $z=0$. $f'(0) = 0$.

Full Solution:

(a) $f(z) = x^2 + iy^2$
$u = x^2$, $v = y^2$.
$u_x = 2x$, $u_y = 0$.
$v_x = 0$, $v_y = 2y$.
Partials are continuous everywhere.
Check CR equations:
1. $u_x = v_y \implies 2x = 2y \implies x = y$.
2. $u_y = -v_x \implies 0 = -0$ (True).
$f'(z)$ exists only when $x=y$.
Value: $f'(z) = u_x + iv_x = 2x + i(0) = 2x$.
So, $f'(x+ix) = 2x$.

(b) $f(z) = z \text{Im}(z)$
$f(z) = (x+iy)y = xy + iy^2$.
$u = xy$, $v = y^2$.
$u_x = y$, $u_y = x$.
$v_x = 0$, $v_y = 2y$.
Partials are continuous everywhere.
Check CR equations:
1. $u_x = v_y \implies y = 2y \implies y = 0$.
2. $u_y = -v_x \implies x = -0 \implies x = 0$.
$f'(z)$ exists only when $x=0$ and $y=0$, i.e., only at $z=0$.
Value: $f'(0) = u_x(0,0) + iv_x(0,0) = 0 + i(0) = 0$.

Q4: Deriving Polar CR Equations

Question:

Derive the polar form of the Cauchy Riemann equations from the Cartesian form.

Final Answer:

Using $x=r\cos\theta, y=r\sin\theta$ and the chain rule for $u_r, u_\theta, v_r, v_\theta$, substitute the Cartesian CR equations $u_x=v_y, u_y=-v_x$ into the expressions for $v_r, v_\theta$ and compare with $u_r, u_\theta$ to derive $u_r = \frac{1}{r}v_\theta$ and $v_r = -\frac{1}{r}u_\theta$.

Full Solution:

Cartesian CR equations: $u_x = v_y$ (1), $u_y = -v_x$ (2).
Coordinate transformation: $x = r\cos\theta$, $y = r\sin\theta$.
Chain rule: $$u_r = u_x x_r + u_y y_r = u_x (\cos\theta) + u_y (\sin\theta)$$ $$u_\theta = u_x x_\theta + u_y y_\theta = u_x (-r\sin\theta) + u_y (r\cos\theta)$$ $$v_r = v_x x_r + v_y y_r = v_x (\cos\theta) + v_y (\sin\theta)$$ $$v_\theta = v_x x_\theta + v_y y_\theta = v_x (-r\sin\theta) + v_y (r\cos\theta)$$ Substitute CR equations (1) and (2) into $v_r$ and $v_\theta$: $$v_r = (-u_y)(\cos\theta) + (u_x)(\sin\theta) = u_x\sin\theta - u_y\cos\theta$$ $$v_\theta = (-u_y)(-r\sin\theta) + (u_x)(r\cos\theta) = r(u_y\sin\theta + u_x\cos\theta)$$ Compare $u_r$ with $v_\theta$: $$u_r = u_x\cos\theta + u_y\sin\theta$$ $$\frac{1}{r}v_\theta = u_y\sin\theta + u_x\cos\theta$$ Thus, $u_r = \frac{1}{r}v_\theta$.

Compare $v_r$ with $u_\theta$: $$v_r = u_x\sin\theta - u_y\cos\theta$$ $$u_\theta = -r u_x \sin\theta + r u_y \cos\theta$$ $$-\frac{1}{r}u_\theta = -(-\frac{1}{r} r u_x \sin\theta + \frac{1}{r} r u_y \cos\theta) = u_x\sin\theta - u_y\cos\theta$$ Thus, $v_r = -\frac{1}{r}u_\theta$.

The polar CR equations are: $u_r = \frac{1}{r}v_\theta$, $v_r = -\frac{1}{r}u_\theta$.

Q5: $f'(z)$ using Polar CR Equations

Question:

Use polar form of Cauchy-Riemann (CR) equations to find $f'(z)$ where :

(a) $f(z) = \sqrt{r} e^{i\theta/2}, (r > 0, \alpha < \theta < \alpha + 2\pi)$

(b) $f(z) = e^{-\theta}\cos(\ln r) + ie^{-\theta}\sin(\ln r), (r > 0, 0 < \theta < 2\pi)$

Final Answer:

(a) $f'(z) = \frac{1}{2\sqrt{r}}e^{-i\theta/2}$

(b) $f'(z) = \frac{i}{z} f(z) = \frac{i e^{-\theta(1+i)} e^{i\ln r}}{r}$

Full Solution:

(a) $f(z) = \sqrt{r} e^{i\theta/2}$
$u = \sqrt{r}\cos(\theta/2)$, $v = \sqrt{r}\sin(\theta/2)$.
$u_r = \frac{1}{2\sqrt{r}}\cos(\theta/2)$, $u_\theta = -\frac{\sqrt{r}}{2}\sin(\theta/2)$.
$v_r = \frac{1}{2\sqrt{r}}\sin(\theta/2)$, $v_\theta = \frac{\sqrt{r}}{2}\cos(\theta/2)$.
Check CR: $u_r = \frac{1}{r}v_\theta \implies \frac{\cos(\theta/2)}{2\sqrt{r}} = \frac{1}{r}\frac{\sqrt{r}\cos(\theta/2)}{2}$ (True).
$v_r = -\frac{1}{r}u_\theta \implies \frac{\sin(\theta/2)}{2\sqrt{r}} = -\frac{1}{r}(-\frac{\sqrt{r}\sin(\theta/2)}{2})$ (True).
Derivative $f'(z) = e^{-i\theta}(u_r + iv_r)$: $$ f'(z) = e^{-i\theta} \left( \frac{\cos(\theta/2)}{2\sqrt{r}} + i \frac{\sin(\theta/2)}{2\sqrt{r}} \right) $$ $$ f'(z) = \frac{e^{-i\theta}}{2\sqrt{r}} (\cos(\theta/2) + i\sin(\theta/2)) = \frac{e^{-i\theta}}{2\sqrt{r}} e^{i\theta/2} $$ $$ f'(z) = \frac{1}{2\sqrt{r}} e^{-i\theta/2} $$

(b) $f(z) = e^{-\theta}\cos(\ln r) + ie^{-\theta}\sin(\ln r)$
$f(z) = e^{-\theta}e^{i\ln r}$.
$u = e^{-\theta}\cos(\ln r)$, $v = e^{-\theta}\sin(\ln r)$.
$u_r = -\frac{e^{-\theta}}{r}\sin(\ln r)$, $u_\theta = -e^{-\theta}\cos(\ln r)$.
$v_r = \frac{e^{-\theta}}{r}\cos(\ln r)$, $v_\theta = -e^{-\theta}\sin(\ln r)$.
Check CR: $u_r = \frac{1}{r}v_\theta \implies -\frac{e^{-\theta}\sin(\ln r)}{r} = \frac{1}{r}(-e^{-\theta}\sin(\ln r))$ (True).
$v_r = -\frac{1}{r}u_\theta \implies \frac{e^{-\theta}\cos(\ln r)}{r} = -\frac{1}{r}(-e^{-\theta}\cos(\ln r))$ (True).
Derivative $f'(z) = e^{-i\theta}(u_r + iv_r)$: $$ f'(z) = e^{-i\theta} \left( -\frac{e^{-\theta}}{r}\sin(\ln r) + i \frac{e^{-\theta}}{r}\cos(\ln r) \right) $$ $$ f'(z) = \frac{e^{-i\theta}e^{-\theta}}{r} (-\sin(\ln r) + i\cos(\ln r)) $$ $$ f'(z) = \frac{e^{-\theta(1+i)}}{r} [i(\cos(\ln r) + i\sin(\ln r))] $$ $$ f'(z) = \frac{i e^{-\theta(1+i)}}{r} e^{i\ln r} $$ (Alternatively, note $f(z)=e^{i\ln z}$, so $f'(z) = e^{i\ln z} \cdot \frac{i}{z} = f(z)\frac{i}{z}$.)

Q6: Differentiability Set for $2x^2+y+i(y^2-x)$

Question:

Find the set of points at which the given function is differentiable :
$f(z) = 2x^2 + y + i(y^2 - x)$

Final Answer:

The function is differentiable only on the line $y=2x$.

Full Solution:

$u = 2x^2 + y$, $v = y^2 - x$.
$u_x = 4x$, $u_y = 1$.
$v_x = -1$, $v_y = 2y$.
Partials are continuous everywhere.
Check CR equations:
1. $u_x = v_y \implies 4x = 2y \implies y = 2x$.
2. $u_y = -v_x \implies 1 = -(-1) \implies 1 = 1$ (True).
Both CR equations hold if and only if $y=2x$.
The set of points where $f$ is differentiable is $\{z = x+iy \mid y = 2x\}$.

Q7: Finding Constants for Analyticity

Question:

Find real constants a and b so that the given function is differentiable everywhere:
$f(z) = 3x - y + 5 + i(ax + by - 3)$

Final Answer:

$a=1$, $b=3$.

Full Solution:

$u = 3x - y + 5$, $v = ax + by - 3$.
$u_x = 3$, $u_y = -1$.
$v_x = a$, $v_y = b$.
Partials are constants, hence continuous everywhere.
For $f$ to be differentiable everywhere, CR equations must hold everywhere:
1. $u_x = v_y \implies 3 = b$.
2. $u_y = -v_x \implies -1 = -a \implies a = 1$.
Thus, we need $a=1$ and $b=3$.

Q8: CR Equations vs Differentiability

Question:

Show that the function $f(z) = \sqrt{|\bar{z}^2 - z^2|}$ satisfies the CR equations at the origin, but is not differentiable. Explain.

Final Answer:

$f(z) = 2\sqrt{|x||y|}$. The partial derivatives $u_x, u_y, v_x, v_y$ are all 0 at (0,0), satisfying CR equations. However, the limit $\lim_{\Delta z \to 0} f(\Delta z)/\Delta z$ depends on the path, showing non-differentiability. This occurs because the partial derivatives are not continuous at (0,0), which is required for the CR equations to be sufficient for differentiability.

Full Solution:

First, $f(z) = \sqrt{|(x-iy)^2 - (x+iy)^2|} = \sqrt{|-4ixy|} = 2\sqrt{|x||y|}$.
$u(x,y) = 2\sqrt{|x||y|}$, $v(x,y) = 0$.

Check CR at origin (0,0):
$u_x(0,0) = \lim_{h\to 0} \frac{u(h,0)-u(0,0)}{h} = \lim_{h\to 0} \frac{0-0}{h} = 0$.
$u_y(0,0) = \lim_{k\to 0} \frac{u(0,k)-u(0,0)}{k} = \lim_{k\to 0} \frac{0-0}{k} = 0$.
$v_x(0,0) = 0$, $v_y(0,0) = 0$.
CR equations: $u_x=v_y \implies 0=0$ (True), $u_y=-v_x \implies 0=-0$ (True).
The CR equations are satisfied at the origin.

Check differentiability at origin:
$f'(0) = \lim_{\Delta z \to 0} \frac{f(\Delta z) - f(0)}{\Delta z} = \lim_{\Delta z \to 0} \frac{2\sqrt{|\Delta x||\Delta y|}}{\Delta x + i\Delta y}$
Approach along $\Delta y = m \Delta x$ ($\Delta z = \Delta x(1+im)$): $$ \lim_{\Delta x \to 0} \frac{2\sqrt{|\Delta x||m\Delta x|}}{\Delta x(1+im)} = \lim_{\Delta x \to 0} \frac{2\sqrt{|m|}|\Delta x|}{\Delta x(1+im)} $$ This limit is $\frac{\pm 2\sqrt{|m|}}{1+im}$ depending on the sign of $\Delta x$. Since the limit depends on $m$ and the direction, the limit does not exist uniquely.
Thus, $f(z)$ is not differentiable at the origin.

Explanation: The Cauchy-Riemann equations being satisfied at a point is a necessary condition for differentiability, but not sufficient. Sufficiency requires the partial derivatives ($u_x, u_y, v_x, v_y$) to also be continuous at the point. Here, while $u_x(0,0)=0$, $u_x(x,y) = \sqrt{|y/x|} \cdot \text{sgn}(x)$ (for $x\ne 0$) which is not continuous at (0,0). The lack of continuity of partial derivatives allows the CR equations to hold while the function is still non-differentiable.

Q9: $\partial / \partial \bar{z}$ Operator and CR Equations

Question:

Define the operator $\frac{\partial}{\partial \bar{z}} = \frac{1}{2}(\frac{\partial}{\partial x} + i \frac{\partial}{\partial y})$. Show that a function $f$ satisfies CR equations if and only if $\frac{\partial f}{\partial \bar{z}} = 0$.

Final Answer:

Applying the operator $\frac{\partial}{\partial \bar{z}}$ to $f=u+iv$ gives $\frac{1}{2}((u_x-v_y) + i(v_x+u_y))$. This expression is zero if and only if $u_x-v_y=0$ and $v_x+u_y=0$, which are the CR equations.

Full Solution:

Let $f(z) = u(x,y) + iv(x,y)$. Apply the operator: $$ \frac{\partial f}{\partial \bar{z}} = \frac{1}{2} \left( \frac{\partial (u+iv)}{\partial x} + i \frac{\partial (u+iv)}{\partial y} \right) $$ $$ \frac{\partial f}{\partial \bar{z}} = \frac{1}{2} \left( (u_x + iv_x) + i (u_y + iv_y) \right) $$ $$ \frac{\partial f}{\partial \bar{z}} = \frac{1}{2} (u_x + iv_x + iu_y + i^2 v_y) $$ $$ \frac{\partial f}{\partial \bar{z}} = \frac{1}{2} (u_x - v_y + i(v_x + u_y)) \quad (*) $$ ($\Rightarrow$) Assume $f$ satisfies CR equations: $u_x = v_y$ and $u_y = -v_x$. Substitute into (*): $$ \frac{\partial f}{\partial \bar{z}} = \frac{1}{2} ( (v_y - v_y) + i(v_x + (-v_x)) ) = \frac{1}{2}(0 + i0) = 0 $$ ($\Leftarrow$) Assume $\frac{\partial f}{\partial \bar{z}} = 0$. From (*), this means: $$ \frac{1}{2} (u_x - v_y) = 0 \implies u_x = v_y $$ $$ \frac{1}{2} (v_x + u_y) = 0 \implies u_y = -v_x $$ These are the CR equations. Therefore, $f$ satisfies CR equations if and only if $\frac{\partial f}{\partial \bar{z}} = 0$.
